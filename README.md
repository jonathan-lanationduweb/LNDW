# LNDW

Site vitrine **La Nation du Web** — propulseur de stratégies digitales.

Site statique HTML / CSS / JS.

## Pages
- `La Nation du Web.html` — page d'accueil
- `arsenal.html` — Notre Arsenal (services)
- `la-nation.html` — La Nation (histoire, équipe, FAQ)
- `missions-clients.html` — Missions clients
- `single-project.html` — Détail d'une mission
- `backlinks.html` — Netlinking
- `blog.html` / `article.html` — Journal de blog
- `contact.html` — Contact
- `mentions-legales.html` / `cgv.html` / `cgu.html` — Pages légales

## Structure
- `css/` — feuilles de styles (`base.css`, `sections.css`, `cta.css`, `paths.css`, `legal.css`, `pages.css`)
- `js/` — interactions (`lndw.js`)
- `assets/` — logos, icônes, décorations, textures, photos
